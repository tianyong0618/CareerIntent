# 人社意图本体库
## 一、意图分类规则
### （一）分类逻辑
以人社核心业务场景为一级分类，按用户身份（应届生/在职/失业/退休等）、业务类型（查询/办理/咨询/投诉等）为二级维度，通过关键词匹配（如“社保缴费”“失业补贴”“劳动仲裁”）界定意图归属，确保分类精准且覆盖全场景。

### （二）一级分类及核心关键词
| 一级分类       | 核心关键词                                                                 |
|----------------|--------------------------------------------------------------------------|
| 就业服务       | 求职、招聘、职业培训、就业指导、实习、应届生就业、灵活就业、失业再就业       |
| 社会保障（社保）| 社保参保、缴费、转移、查询、补缴、停保、续保、社保基数、社保待遇             |
| 社会保障（医保）| 医保报销、异地就医、医保缴费、医保查询、定点医院、药品目录、门诊慢特病       |
| 住房公积金     | 公积金提取、缴存、转移、查询、贷款、还款、公积金基数                         |
| 劳动关系       | 劳动合同、工资、加班、辞退、离职、劳动仲裁、工伤认定、经济补偿               |
| 补贴福利       | 应届生补贴、创业补贴、失业补贴、租房补贴、育儿补贴、高温补贴、残疾人补贴     |
| 退休养老       | 退休办理、养老金、退休资格认证、延迟退休、养老待遇、视同缴费年限             |
| 人才服务       | 人才引进、落户、职称评定、人才补贴、学历认证、技能证书、职业资格               |
| 人事档案       | 档案查询、档案转移、档案托管、档案补办、档案政审、学籍档案、工作档案           |

## 三、求职 / 培训 / 政策咨询等 5 类意图
1. 测评规则：如 “技术岗能力维度 = 编程能力 + 沟通能力 + 项目经验”“性格内向→适配岗位 = 后端开发 / 数据工程师”；
2. 政策信息：如 “东城区应届生补贴 = 5000 元，条件 = 应届生 + 东城区就业，流程 = 线上申请→审核→到账”；
3. 能力缺口对应课程：如 “沟通能力弱→推荐《STAR 法则实战课》”“面试表达差→《技术面试话术指南》”
